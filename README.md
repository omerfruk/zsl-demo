# Zero Shot Learning

## Installation

Create a virtual environment and install the requirements
```bash
python -m venv .venv 
```
Activate the virtual environment
```bash
source .venv/bin/activate
```

For Mac/Linux
```bash
pip3 install -r requirements.txt
```

For Windows
```bash
pip install -r requirements.txt
```

## Usage
For Mac/Linux
```bash
python3 main.py
```
For Windows
```bash
python main.py
```
